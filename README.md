# RKdoces

Site institucional estatico para divulgacao de bolos, doces e salgados artesanais.

## Estrutura

- `index.html`: pagina principal com destaque da marca e acesso ao catalogo.
- `css/estilo.css`: identidade visual compartilhada entre todas as paginas.
- `paginas/`: secoes de bolos, doces, salgados e atendimento.
- `img/`: imagens utilizadas no catalogo e nas secoes promocionais.

## Objetivo

Apresentar a marca `RKdoces` com uma imagem mais profissional, textos comerciais mais claros
e navegacao simples para facilitar o atendimento pelo WhatsApp.
